FScan (Port Scanning)
  Input: ports with IP, port, service, version
  Output: Vulnerability records with severity="info"
  Display: Table with IP, Port, Service, Version

Nuclei (Vulnerability Scanning)
  Input: vulnerabilities with name, severity, URL
  Output: Vulnerability records with extracted severity
  Display: Table with Name, Severity, Matched URL

Afrog (PoC Execution)
  Input: findings with vulnerability, severity, target
  Output: Vulnerability records with severity
  Display: Table with Name, Severity, Target

DDDD (Advanced Scanning)
  Input: advanced findings with name, severity
  Output: Vulnerability records
  Display: Table with Name, Severity, Details

DirSearch (Directory Enumeration)
  Input: directories with path, HTTP status
  Output: Vulnerability records with severity="info"
  Display: Table with Path, HTTP Status
